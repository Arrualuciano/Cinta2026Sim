Detailed Description

Implementacion del clasificador de cajas para cinta transportadora.

Logica de clasificacion basada en una unica lista de cajas en transito. Cada caja lleva su propio campo stage (proximo sensor esperado), lo que permite razonar sobre cada caja en forma independiente y evita las desincronizaciones tipicas de los esquemas con varias colas FIFO paralelas.

Invariantes de la lista:

  • Las cajas estan ordenadas por orden de llegada (la mas antigua en list[0]).
  • El campo stage de cada caja solo avanza (nunca retrocede).
  • Una caja se elimina unicamente cuando es expulsada por un actuador o cuando supera el ultimo sensor sin ser clasificada.

◆ Classifier_OnSensor()

void Classifier_OnSensor ( classifier_t * clas,
uint8_t outNum )

Notifica al clasificador que el sensor de la salida outNum detecto una caja.

Busca la caja mas antigua cuyo stage coincida con outNum y la procesa:

  • Si el tipo coincide con boxType[outNum] y el actuador acepta el disparo: la caja es expulsada y eliminada de la lista.
  • En cualquier otro caso (tipo incorrecto o brazo ocupado): se avanza el stage para que la caja siga siendo rastreada hacia la proxima salida.
  • Si el stage supera el ultimo sensor, la caja se descarta (cayo al final).
Parameters
clasPuntero al clasificador.
outNumNumero de salida cuyo sensor disparo (0, 1 o 2).
