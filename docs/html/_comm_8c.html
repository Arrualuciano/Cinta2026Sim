Detailed Description

Implementacion del modulo de comunicacion serie con protocolo UNER.

Los buffers circulares de RX y TX son arreglos estaticos de 128 bytes (potencia de 2) declarados en este archivo; sus punteros se asignan a la estructura Comm_Protocol durante Comm_Init().

Configuracion de UART0 (Comm_InitUART):

  • F_CPU = 16 MHz, U2X = 1, UBRR = 16 --> 115200 baud
  • Formato: 8N1
  • Interrupciones: RX Complete habilitada

◆ CommFsmState_t

enum CommFsmState_t

Estados internos de la FSM de decodificacion de tramas UNER.

La FSM sincroniza con el encabezado "UNER", luego captura el campo de longitud, el separador ':' y finalmente el payload byte a byte hasta verificar el checksum.

◆ Comm_Process()

void Comm_Process ( Comm_Protocol * comm)

Procesa el modulo de comunicacion: envio y recepcion.

Debe llamarse periodicamente desde el main loop (no bloqueante).

  • TX: transmite un byte pendiente si el registro UDR0 esta libre.
  • RX: corre la FSM UNER sobre todos los bytes disponibles en el buffer de recepcion e invoca el callback onCmd al completar una trama valida con checksum correcto.
Parameters
commPuntero a la instancia Comm_Protocol.
