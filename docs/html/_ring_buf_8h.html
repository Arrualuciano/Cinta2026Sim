Detailed Description

Buffer circular (ring buffer) de proposito general.

Estructura de datos para buffers circulares de tamano fijo. El tamano DEBE ser potencia de 2 para que el mascareo con (size - 1) funcione correctamente como reemplazo del modulo.

Escritura (ISR o productor):

buf[iw++] = dato;
iw &= (size - 1);

Lectura (main loop o consumidor):

while (ir != iw) {
uint8_t b = buf[ir++];
ir &= (size - 1);
}
Note
iw e ir son uint8_t, por lo que el tamano maximo util es 128 (potencia de 2, menor que 256).
