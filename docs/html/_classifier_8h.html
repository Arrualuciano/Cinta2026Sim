Detailed Description

Clasificador de cajas para cinta transportadora.

Mantiene una lista ordenada de cajas en transito en la cinta. Cada caja lleva un campo stage que indica el proximo sensor que se espera la detecte, lo que permite razonar sobre cada caja de forma independiente y evitar desincronizaciones tipicas de esquemas con multiples colas FIFO paralelas.

Flujo de uso:

// Al iniciar (o tras RESET):
Classifier_Init(&clas, boxTypes, onTrigger);
// Cuando el simulador mide una nueva caja:
Classifier_NewBox(&clas, tipo);
|
// Cuando un sensor IR detecta la entrada de una caja:
Classifier_OnSensor(&clas, numSalida);

La logica de despacho en Classifier_OnSensor():

  1. Busca la caja mas antigua cuyo stage coincida con outNum.
  2. Si el tipo coincide con la salida y el actuador esta libre: dispara y elimina la caja.
  3. Si no coincide o el actuador esta ocupado: avanza el stage de la caja.
  4. Si stage supera el ultimo sensor: la caja cae al final de la cinta y se descarta.

◆ Classifier_OnSensor()

void Classifier_OnSensor ( classifier_t * clas,
uint8_t outNum )

Notifica al clasificador que el sensor de la salida outNum detecto una caja.

Busca la caja mas antigua cuyo stage coincida con outNum y la procesa:

  • Si el tipo coincide con boxType[outNum] y el actuador acepta el disparo: la caja es expulsada y eliminada de la lista.
  • En cualquier otro caso (tipo incorrecto o brazo ocupado): se avanza el stage para que la caja siga siendo rastreada hacia la proxima salida.
  • Si el stage supera el ultimo sensor, la caja se descarta (cayo al final).
Parameters
clasPuntero al clasificador.
outNumNumero de salida cuyo sensor disparo (0, 1 o 2).
